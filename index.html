Repulser: Invasion of the Aspect

Repulser is an old school, dos-styled shmup. Take on the Aspect with a selection of 3 different ships, 3 upgradeable weapons and an experimental beam super weapon.

Zombie Apocalyp-tic-80 is a prototype arena shooter developed for the open-source Tic-80 fantasy console. Choose a weapon and blast your way through 10 levels. Earn talents and collect upgrades from fallen foes. Play by clicking here.

SpaceTrade Ascii Edition

SpaceTrader is a small project I developed to learn the basics of C#. It uses the console as a canvas and draws in real time. It came about from wanting to make 'text' space game like elite and thought Ascii graphics would be more interesting.

The engine could be used to make other games such as platformers, shooters etc. It uses the .Net framework with no additional libraries. Currently I am developing a full version of SpaceTrade with normal graphics using Monogame.

Raycasting 3D Engine

This engine was intended to be used for a dungeon crawler game that never saw the light of day.

It is adapted from Lode's cg tutorials. They are fantastic tutorials and found them very insightful into implementing this "3D" rendering method. Lode's examples are all in C++ but it was straight forward enough porting it to C#.
